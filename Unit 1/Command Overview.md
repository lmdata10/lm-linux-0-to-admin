
# L2: Using Essential Tools

```bash
whoami                    # Shows your username
ls                        # Lists files in current directory
ls -la                    # Lists all files with details
ip a                      # Shows network interfaces and IPs
cat file.txt              # Displays file contents
passwd                    # Changes your password
touch newfile.txt         # Creates empty file
pwd                       # Shows current directory path
man ls                    # Opens manual for 'ls' command
apropos network           # Searches commands about 'network'
man -k network            # Same as apropos
sudo mandb                # Updates manual database
pinfo bash                # Opens info documentation for bash
tldr tar                  # Shows simple examples for 'tar'
```
